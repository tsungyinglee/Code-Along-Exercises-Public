# This code demonstrates nested loops with a break statement.
# For each value of i from 0 to 3, j also goes from 0 to 3.
# When j becomes greater than i, the inner loop breaks.
# This results in printing all pairs (i, j) where j <= i.
# Outer loop: i goes from 0 to 3
for i in range(4):
    # Inner loop: j goes from 0 to 3
    for j in range(4):
        # If j is greater than i, exit the inner loop
        if j > i:
            break
        # Print the current values of i and j
        print(i, j)

# This code demonstrates a while loop that halves the value of x in each iteration,
# adds it to the running total, and stops if the total exceeds 500.
# The loop starts with x = 256 and continues halving x (using integer division)
# until x becomes 0 or the total exceeds 500.

# Initialize x with 256
x = 256
# Initialize total with 0
total = 0
# Start the while loop; continue as long as x > 0
while x > 0:
    # If total exceeds 500, exit the loop
    if total > 500:
        break
    # Add current x to total
    total += x
    # Halve x using integer division
    x = x // 2
# Output the final total

print("x =", x, "total =", total)
